.. _Query:

Query

    **Attributes:**

        | **page**: Optional[int]
        | **amount**: Optional[int]
        | **fields**: Optional[list[str]]

        | **orderBy**: Optional[list["asc" or "desc"]]

    .. code-block::
        :caption: **Example:**

            orderBy = {
                "page": 1,
                "fields": ["id", "name"],
                "filter": {"name": "test"},
                "amount": 20,
                "orderBy": ["name": "asc"]
            }
